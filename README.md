## 🧩 Current Modules

### 👁️ Monocular  
**Purpose:** TCP port scanner + banner grabber  
**Features:**
- Targeted port scanning with CLI controls
- Banner grabbing using stealth browser-like headers
- Randomized delays for low-detection recon
- Status output with timing data per port

---

### 🕊️ Perch  
**Purpose:** Subdomain enumeration  
**Features:**
- Fast DNS resolution-based subdomain enumeration
- Custom wordlist support
- Quiet mode with only successful hits
- Designed to feed directly into Monocular

---

### 🔧 Clutch  
**Purpose:** Web directory brute-forcer  
**Features:**
- URL fuzzing with wordlist input
- Optional file extension support
- Status code filtering, response length diffing
- Random delay + header rotation for stealth
- Quiet mode for minimal detection footprint
